# KebabVPN — лендинг VPN-сервиса

Промо-лендинг для концепта VPN-сервиса: первый экран, навигация, секции с преимуществами и призыв к действию.

**Живое демо:** https://geregor.github.io/KebabVPN/

## Что показывает проект

- Структура лендинга: `header` / `nav` / `main` / `footer`
- Секции с преимуществами и CTA-кнопки
- Кастомная типографика (Montserrat, Advent Pro, Roboto через Google Fonts)

## Стек

- HTML5, CSS3
- Google Fonts

## Примечание

Учебно-практический проект по вёрстке, только фронтенд. Один из серии лендингов.
